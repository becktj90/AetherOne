
# AetherSense production UI update

This update pushes the web experience toward a more production-ready AetherSense brand language.

## What changed
- Replaced remaining top-level SniffMaster branding in the main web UI.
- Shifted nav labels to a more premium vocabulary: Overview, Atmosphere, Signals, Trends, Hardware, Experiments.
- Reframed odor/stank language toward signal classification and air index terminology.
- Tightened mobile behavior and topbar polish.
- Updated manifest metadata to AetherSense branding.

## Recommended next passes
- Replace deeper legacy variable names in firmware and API payloads only after confirming no backend regressions.
- Align event tags and history labels with the same vocabulary.
- Add compact mobile cards for the hero status row.
